## Admin Panel

###### Pending Reviews
<p style="font-size: 10px; font-weight: 1;">Here privileged users can inspect images and either approve or reject them for display on galleries</small>

![Screenshot](https://raw.githubusercontent.com/Memtly/Memtly.Assets/master/Screenshots/Community/Tablet/Light/AdminPanel_ReviewsTab.png)

###### Gallery List
<p style="font-size: 10px; font-weight: 1;">Here privileged users can view a list of galleries and perform actions on them</small>

![Screenshot](https://raw.githubusercontent.com/Memtly/Memtly.Assets/master/Screenshots/Community/Tablet/Light/AdminPanel_GalleriesTab.png)

###### Users List
<p style="font-size: 10px; font-weight: 1;">Here privileged users can view a list of users and perform actions on them</small>

![Screenshot](https://raw.githubusercontent.com/Memtly/Memtly.Assets/master/Screenshots/Community/Tablet/Light/AdminPanel_UsersTab.png)

###### Custom Resourse List
<p style="font-size: 10px; font-weight: 1;">Here privileged users can view a list of custom resources used to customise the site and perform actions on them</small>

![Screenshot](https://raw.githubusercontent.com/Memtly/Memtly.Assets/master/Screenshots/Community/Tablet/Light/AdminPanel_ResourcesTab.png)

###### Settings Page
<p style="font-size: 10px; font-weight: 1;">Here privileged users can modify the site settings to enable and disable specific features or change to overall look and feel of the site</small>

###### Audit List
<p style="font-size: 10px; font-weight: 1;">Here privileged users can view a list of all actions performed by other users</small>

![Screenshot](https://raw.githubusercontent.com/Memtly/Memtly.Assets/master/Screenshots/Community/Tablet/Light/AdminPanel_AuditTab.png)
