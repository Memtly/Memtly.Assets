WeddingShare is now Memtly

We've rebranded. The project, community, codebase and sponsors links have all moved to a new home under the Memtly organisation.

Notice: Memtly is not directly upgradable from WeddingShare! As Memtly is a major redesign with both frontent and backend changes there are breaking changes between the two images. For the full upgrade path please visit the Migration Guide for more information

Documentation has been set up to reflect the changes

The BuyMeACoffee link has changed - https://buymeacoffee.com/memtly

Why rebrand to Memtly?

As more and more people started using WeddingShare we quickly realised that people were using it for more than just weddings. We've seen concerts, friend trips, and a whole host of other events. It only felt right that we rebrand to something more "generic" to cover all bases and reduce confusion when inevitably someone recommends it to a friend and their response is "but I'm not getting married". That's where Memtly comes in... Now along with the improved look, we also have an improved name.
